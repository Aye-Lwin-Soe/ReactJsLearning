# ReactJsLearning

A simple React.js learning project for practicing React fundamentals, components, state management, props, hooks, routing, and API integration.

## Requirements

* Node.js >= 18
* npm >= 9

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/ReactJsLearning.git
cd ReactJsLearning
```

Install dependencies:

```bash
npm install
```

## Running the Project

Start the development server:

```bash
npm start
```

or

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:3000
```

## Project Structure

```text
src/

```
